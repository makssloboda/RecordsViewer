Проект представляє собою додаток для створення, редагування та зберігання даних в вигляді
деревовидної структури, де кожен вузол дерева є папкою або записом.
Додаток було реалізовано на платформі .NET 5.0.1 за допомогою фреймворку WPF та мови програмування C#.
Для побудови архітектури додатку було застосовано шаблон проектування Model-View-ViewModel, що 
дозволило відокремити графічний інтерфейс додатку від бізнес-логіки. Струтуру додатку було 
реалізовано відповідно до шаблону проектування Компонувальник, це дозволило полегшити звернення
до окремих вузлів дерева та з легкістю здійснювати маніпуляції над вузлами. Для написання юніт-тестів 
було використано фреймворк xUnit. Також, дерево містить тестові дані для демонстрації роботи додатку.

Опис функцій додатку:

1. Створення нових вузлів (папок та записів). 
Створити нову папку чи запис в кореневому каталозі можна за допомогою пункту меню 'Add'. 
При створенні нової папки, користувачу пропонується одразу задати ім'я папки. Щоб застосувати зміни 
достатньо перевести фокус на інший елемент інфтерфейсу додатку. При створенні нового запису потрібно
надати інформацію про запис, скориставшись відповідними полями для вводу. Для застосування змін можна 
натиснути на кнопку з іконкою галки або натиснути клавішу Enter. Створити папку чи запис можна також 
в уже існуючих папках, для цього необхідно натиснути на відповідну папку правою кнопкою миші, та обрати
пункт 'Add record' або 'Add folder' у контексному меню.

2. Переглядання та редагування даних користувачів. 
Щоб переглянути детальні дані про користувача, потрібно обрати відповідного користувача в дереві
одинарним натисненням на ліву кнопку миші. Після цього на екран буде виведено детальну інформацію про
користувача, яку можна відредагувати. Для редагування потрібно натиснути на кнопку з іконкою олівця
біля відповідного інформаційного поля. Після цього, поле перейде у стан редагування і поле можна буде 
змінити. Застосувати зміни можна декількома способами: натиснути клавішу Enter; перевести фокус на інший 
елемент графічного інтерфейсу додатку; натиснути на кнопку з іконкою галки біля відповідного поля.

3. Пошук користувачів та папок за назвою. 
Знайти потрібного користувача можна за допомогою поля для вводу внизу графічного інтерфейсу додатку.
Для цього необхідно ввести ім'я відповідного користувача та натиснути кнопку з іконкою пошукової лупи
або кнопку Enter. Схожим чином виконується і пошук папок. Якщо відповідного користувача буде знайдено,
його буде виділено у дереві, та на екран виведеться детальна інформація користувача. При здійсненні 
пошуку папок, відповідну папку буде виділено у дереві. Якщо запис не буде знайдено, пошуковий запит 
буде проігноровано.

4. Пошук за шаблоном та фільтрація пошуку. 
Для спрощення здійснення пошуку було реалізовано пошук за шаблоном та фільтрацію пошуку. Знайти 
потрібного користувача можна ввівши лише по одній першій букві його імені та фамілії. При цьому, 
реєстр символів буде ігноруватись. Якщо результатів пошуку опинеться декілька, то додаток відобразить
той, що було знайдено швидше. Також, підтримується фільтрація папок та користувачів. Щоб здійснити пошук 
лише папок потрібно додати символи 'f:' або 'F:' перед пошуковим запитом. Для здійснення пошуку лише по
користувачах потрібно додати символи 'r:' або 'R:'. Пошук за шаблоном та фільтрацію пошуку можна комбінувати,
наприклад, наступний запис можна використати для пошуку користувача з іменем Joe Markov: 'r: JM'.

5. Видалення записів та папок, перейменування папок.
Кожен вузол дерева при необхідності можна виділити. Для цього потрібно натиснути на відповідний вузол 
правою кнопкою миші та обрати пункт 'Delete' у контексному меню. При видаленні папок також будуть виділені 
усі папки та записи, що знаходяться у папці що видаляється. Також, папки можна перейменовувати, для цього 
потрібно обрати пункт 'Rename' у контексному меню відповідної папки. Після цього, поле назви папки 
відкриється на редагування і назву можна буде змінити. Щоб застосувати зміни, достатньо перевести фокус на
інший елемент графічного інтерфейсу додатку.
